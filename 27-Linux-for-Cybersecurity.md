# 27-Linux-for-Cybersecurity

## Introduction

Linux is the most widely used operating system in cybersecurity. It powers servers, cloud platforms, security tools, and penetration testing environments. Most cybersecurity professionals use Linux because of its flexibility, stability, powerful command-line interface, and open-source nature.

Whether you work in SOC, GRC, Penetration Testing, DFIR, or Cloud Security, Linux skills are essential.

---

# Why is Linux Important in Cybersecurity?

Linux is used because it:

- Is open-source and customizable
- Has powerful command-line tools
- Is stable and reliable
- Supports thousands of security tools
- Runs most web servers and cloud platforms

# Common Linux Commands for Security

Check logged-in users:

```bash
who
```

View active processes:

```bash
ps aux
```

Check open ports:

```bash
ss -tuln
```

View system logs:

```bash
journalctl
```

Check disk usage:

```bash
df -h
```

View network interfaces:

```bash
ip addr
```

# Best Practices

- Keep your Linux system updated.
- Use strong passwords and SSH keys.
- Disable unnecessary services.
- Monitor logs regularly.
- Follow the Principle of Least Privilege.
- Backup important data.
